Instalando o Odoo
-----------------

A primeira coisa que você precisa entender é que existem várias formas de instalar o Odoo,
algumas mais simples e outras mais complexas. A escolha correta sempre vai depender de alguns fatores:

- Ambiente: desenvolvimento ou produção;
- Nível de automação / arquitetura da cloud;
- Nível de experiência do time;


.. nextslide::

É possível subir o Odoo automaticamente no AWS/GCP através de um pipeline e uma Squad ágil, claro! Mas para
quem esta começando é necessário começar pelas formas mais simples:
